

# 🏋️‍♂️ AuraGym - Fitness & Gym Management Platform

AuraGym is a modern, fully responsive web application designed for fitness enthusiasts. Built with a clean, minimal, and premium aesthetic, it offers a seamless experience for users to explore gym classes, manage schedules, and track their fitness journey.

## ✨ Core Features
- **Secure Authentication:** Complete user login and registration system.
- **Interactive Dashboard:** Dedicated and dynamic dashboard for managing profiles, classes, and activities.
- **Dynamic Data Management:** Real-time loading of class schedules, trainer information, and gym services from the MongoDB database.
- **Fully Responsive Design:** Optimized for smooth performance across desktop, tablet, and mobile devices.
- **Premium UI/UX:** A clutter-free, professional user interface leveraging modern styling.

## 🛠️ Technology Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Deployment:** Vercel (Both Client and Server)
